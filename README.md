## Notes
* Built using [`vue-cli`](https://github.com/vuejs/vue-cli) (Vue CLI 3)

## Quickstart
Start & install dependencies:
```bash
$ npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
